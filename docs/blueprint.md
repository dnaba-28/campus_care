# **App Name**: CareCampus

## Core Features:

- SOS Emergency Alert: Clickable card that simulates sending an emergency alert and updates its text accordingly.
- Campus Hospital Status: Displays the real-time status of the campus hospital, including doctor availability and waiting time.
- Cafeteria Status: Display what food are on the menu at the Cafeteria
- Realtime SOS Analyzer: Use generative AI to listen and respond in the best way to audio reports, providing human feedback on optimal campus resource usage for these emergencies
- Navbar with Responsive Links: Top navigation bar with logo, Home, Admin, and Login links, with mobile-responsive design (hamburger menu).
- Responsive Grid Layout: The core dashboard layout uses a responsive grid, adapting to different screen sizes (1 column on mobile, 3 columns on desktop).

## Style Guidelines:

- Primary color: Medical blue (#4AB5F4) to convey trust and safety.
- Background color: Light blue (#E0F7FA), a very pale tint of the primary.
- Accent color: Turquoise (#4AF4D4) for highlights and interactive elements.
- Font pairing: 'Poppins' (sans-serif) for headers and 'PT Sans' (sans-serif) for body text; to create a contemporary and accessible feel.
- Lucide-React icons are used throughout the application to provide a consistent and clean visual language.
- A max-width container centers the grid, and cards have a gap of 6 or 8 to maintain spacing.
- Pulse animation on the SOS Emergency card to immediately draw attention in critical situations.
